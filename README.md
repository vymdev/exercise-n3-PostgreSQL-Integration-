## More about this exercise

/🇬🇧/ This project has a more realistic and useful feel for real-world projects. I've connected a PostgreSQL database to Spring Boot and created a REST API to query and create products with monetary values. I then validate these values ​​using jakarta.validation.valid to ensure they are not null or erroneous. It's not a full CRUD application; it's more of an experimental exercise to demonstrate my Spring Boot development skills. Thanks for reading and checking out my repository.

/🇪🇸/ este proyecto tiene ya un toque mas realistico y util para proyectos reales, en el que conecto una base de datos de PostgreSQL con Springboot y hago una API Rest para consultar y crear productos con un valor de dinero, Y valido con jakarta.validation.Valid que estos valores no sean nulos ni erroneos, No es un CRUD completo, es mas bien un ejercicio con intencion de ser experimental, es para demostrar mi desarrollo en Springboot, Gracias por leer y revisar mi repositorio.

---

## Features and uses

- REST Endpoints to create and query products.
- Integration with **PostgreSQL** using Spring Data JPA.
- Validation with **jakarta.validation** to ensure product data is correct.
